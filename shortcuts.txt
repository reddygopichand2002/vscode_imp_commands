Ctrl + Shift + F: Open the Search panel
Ctrl + Shift + H: Replace all occurrences of a word in the current file
Ctrl + Shift + L: Select all occurrences of a word in the current file
Ctrl + Shift + ]: Indent selected lines
Ctrl + Z: Undo the last action
Ctrl + A: Select all text in the current file
Ctrl + F: Find text in the current file
Ctrl + G: Go to a specific line in the current file
Ctrl + Shift + I auto-indent your code to make it more readable and consistent
